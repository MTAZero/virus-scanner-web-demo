# Test Files cho Virus Scanner

## EICAR Test File

File `eicar_test.txt` là file test chuẩn của ngành antivirus, được tạo bởi European Institute for Computer Antivirus Research (EICAR).

### Đặc điểm:
- ✅ **Hoàn toàn an toàn** - Không phải virus thật
- ✅ **Được nhận diện bởi tất cả antivirus** - Dùng để test phần mềm antivirus
- ✅ **File text đơn giản** - Có thể mở bằng text editor

### Nội dung file:
```
X5O!P%@AP[4\PZX54(P^)7CC)7}$EICAR-STANDARD-ANTIVIRUS-TEST-FILE!$H+H*
```

### Cách sử dụng:
1. Upload file `eicar_test.txt` lên web scanner
2. File sẽ được phát hiện như một "virus" (đây là hành vi mong đợi)
3. Đây là cách an toàn để test scanner mà không cần file virus thật

## Lưu ý:
- **KHÔNG** download file virus thật từ internet
- **KHÔNG** sử dụng malware thật để test (có thể gây hại)
- Chỉ sử dụng file test như EICAR hoặc các file mẫu từ nguồn uy tín

## Nguồn tham khảo:
- EICAR: https://www.eicar.org/
- Các file test khác có thể tìm trên các nền tảng sandbox như VirusTotal, Any.run (chỉ xem, không download)
